GAMEIN

Sito web per appassionati di videogiochi.
Possibilità di salvare titoli preferiti in una lista personale.
Funzionalità di sign-in e login per recensire i giochi.
Pagine dedicate ai singoli giochi con descrizioni dettagliate.
Visualizzazione di statistiche e recensioni degli utenti.
Ricerca dinamica con filtri per anno, genere e sviluppatore.




Strumenti utilizzati

Git per il versioning del codice.
Maven per la gestione delle dipendenze.
Bootstrap per un'interfaccia responsive.
Linguaggi utilizzati: JavaScript, JSP, Java, MySQL, HTML/CSS.
